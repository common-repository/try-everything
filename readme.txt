Try Everything, by Shakira, written in the form of inspiration.

== Description ==

The lyrics of Try Everything, performed by Shakira, shown to you to inspire you to continue your work.

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/try-everything directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress or click on 'Active Plugin' once it is installed.
3. Begin to see inspirational words appear in the top-right of the admin page.
